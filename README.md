# Colony

A plugin marketplace for Claude Code. Browse, install, and go.

## Install

Add the marketplace in Claude Code:

```
/plugin marketplace add dhruvil009/colony
```

Then install any plugin:

```
/plugin install hivescanner@colony
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| **hivescanner** | Zero-token background monitoring — watches GitHub, Slack, and more |

Run `/plugin` and go to the **Discover** tab to browse all available plugins.
